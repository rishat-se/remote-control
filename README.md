# Websocket Remote Control

## Installation

1. Clone repo:
   git clone https://github.com/rishat-se/remote-control.git

2. `npm install`

## Usage

**Production**

`npm run start`

-   App served @ `http://localhost:8181` without nodemon

**Development**

`npm run start:dev`

-   App served @ `http://localhost:8181` with nodemon
